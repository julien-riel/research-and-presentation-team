# Principes de Composition Visuelle pour Images Stock

Ce document compile les principes de sélection et d'utilisation d'images photographiques dans les présentations professionnelles.

---

## 1. La Règle des Tiers

### Principe Fondamental

Diviser l'image en 9 parties égales (grille 3x3). Les points d'intersection sont les « points forts » où placer les sujets principaux.

```
┌───────┬───────┬───────┐
│       │   ●   │       │
├───────┼───────┼───────┤
│   ●   │       │   ●   │
├───────┼───────┼───────┤
│       │   ●   │       │
└───────┴───────┴───────┘
      Points forts
```

### Application en Présentation

| Placement | Effet | Usage |
|-----------|-------|-------|
| Sujet au centre | Statique, formel | Portrait corporate |
| Sujet à gauche | Dynamique, appelle le regard vers la droite | Image + texte à droite |
| Sujet à droite | Introspectif | Texte à gauche + image |

---

## 2. Lignes Directrices (Leading Lines)

### Types de Lignes

| Type | Effet Psychologique | Exemple |
|------|---------------------|---------|
| **Horizontales** | Calme, stabilité | Horizon, tables |
| **Verticales** | Force, croissance | Bâtiments, arbres |
| **Diagonales** | Dynamisme, mouvement | Routes, escaliers |
| **Courbes** | Élégance, naturel | Rivières, chemins |

### Usage en Présentation

Les lignes doivent guider l'œil vers :
1. Le titre ou message clé
2. Le call-to-action
3. Le sujet principal

**Éviter** : Lignes qui sortent du cadre ou pointent vers l'extérieur

---

## 3. Espace Négatif (White Space)

### Règle pour les Présentations

> « Une image avec 40-60% d'espace négatif permet d'ajouter du texte
> sans surcharger visuellement. »

### Recherche d'Images Optimales

Privilégier les images avec :
- Zones floues (bokeh) pour le texte
- Ciels unis ou dégradés
- Surfaces monochromes (murs, tables)

### Exemples de Requêtes Pexels

| Besoin | Requête Optimale |
|--------|------------------|
| Texte sur image | `[sujet] with copy space` |
| Fond business | `minimalist office background` |
| Abstrait tech | `abstract technology background` |

## 4. Psychologie des Couleurs

### Dominantes Chromatiques

| Couleur Dominante | Émotion | Secteurs Appropriés |
|-------------------|---------|---------------------|
| **Bleu** | Confiance, professionnalisme | Finance, Tech, Santé |
| **Vert** | Nature, croissance, durabilité | Environnement, Agriculture |
| **Orange** | Énergie, enthousiasme | Startups, Commerce |
| **Violet** | Luxe, créativité | Mode, Beauté |
| **Gris** | Neutralité, sophistication | Industrie, B2B |
| **Blanc** | Pureté, simplicité | Minimalisme, Médical |

### Harmonie avec le Thème

L'image doit s'intégrer à la palette de couleurs de la présentation :

```
Thème bleu corporate (#1E3A5F)
  → Chercher : images à dominante bleue/grise
  → Éviter : images saturées orange/jaune
```

## 5. Authenticité vs Stock Cliché

### Anti-Patterns à Éviter

| Cliché Stock | Pourquoi c'est mauvais | Alternative |
|--------------|------------------------|-------------|
| Poignée de main générique | Vu des millions de fois | Collaboration réelle, mains sur projet |
| Équipe qui saute en l'air | Artificiel | Réunion authentique, sourires naturels |
| Femme qui rit avec salade | Meme d'internet | Photo lifestyle naturelle |
| Homme en costume qui court | Surexploité | Mouvement au travail réel |
| Flèches 3D, graphiques CGI | Daté, kitsch | Graphiques minimalistes 2D |

### Indicateurs d'Authenticité

✅ **Bon** :
- Éclairage naturel
- Expressions non forcées
- Décors réels (pas de studio blanc parfait)
- Diversité des physiques

❌ **Mauvais** :
- Éclairage studio parfait
- Sourires forcés identiques
- Poses non naturelles
- Uniformité des modèles

## 6. Résolution et Qualité

### Standards pour Présentations

| Usage | Résolution Minimum | Recommandée |
|-------|-------------------|-------------|
| Plein écran 1080p | 1920 x 1080 | 2560 x 1440 |
| Plein écran 4K | 3840 x 2160 | Original 4K |
| Demi-slide | 1200 x 800 | 1600 x 1067 |
| Vignette | 600 x 400 | 800 x 533 |

### Compression et Formats

| Format | Usage | Taille Typique |
|--------|-------|----------------|
| **JPEG** | Photos standard | 100-500 KB |
| **PNG** | Photos avec transparence | 500 KB - 2 MB |
| **WebP** | Web optimisé | 50-200 KB |

## 7. Orientation et Ratio

### Landscape (Paysage) - 16:9, 3:2

**Idéal pour** :
- Fonds de slide pleine largeur
- Images panoramiques
- Contexte environnemental

### Portrait - 2:3, 9:16

**Idéal pour** :
- Portraits individuels
- Mise en page deux colonnes
- Stories verticales

### Carré - 1:1

**Idéal pour** :
- Grilles d'équipe
- Comparaisons côte à côte
- Réseaux sociaux

### Recommandations par Contexte

| Contexte de Slide | Orientation | Ratio |
|-------------------|-------------|-------|
| Image de fond pleine | Landscape | 16:9 |
| Image + texte à droite | Portrait ou Carré | 2:3 ou 1:1 |
| Bandeau inférieur | Landscape large | 4:1 |
| Grille 3 images | Carré | 1:1 |

## 8. Droits et Attributions

### Licences Pexels

Pexels utilise une licence libre qui permet :
- ✅ Usage commercial sans attribution
- ✅ Modification des images
- ✅ Usage en présentations, impressions, web

Ne permet PAS :
- ❌ Revente des images brutes
- ❌ Usage dans des produits à la demande (print-on-demand)
- ❌ Représentation négative des personnes identifiables

### Bonnes Pratiques

Même si l'attribution n'est pas obligatoire :
- Créditer le photographe si possible (slide de fin)
- Conserver les métadonnées pour traçabilité
- Ne pas utiliser de personnes identifiables pour des sujets sensibles

## 9. Checklist de Sélection

Avant d'utiliser une image dans une présentation :

### Qualité Technique
- [ ] Résolution suffisante pour l'affichage prévu ?
- [ ] Pas de grain/bruit excessif ?
- [ ] Mise au point correcte sur le sujet ?
- [ ] Pas de filigrane visible ?

### Composition
- [ ] Sujet bien positionné (règle des tiers) ?
- [ ] Espace pour ajouter du texte si nécessaire ?
- [ ] Lignes directrices favorables ?

### Message
- [ ] L'image renforce le propos ?
- [ ] Pas de cliché stock évident ?
- [ ] Authenticité des expressions/poses ?

### Cohérence
- [ ] Palette de couleurs compatible avec le thème ?
- [ ] Style cohérent avec les autres images ?
- [ ] Orientation adaptée au layout ?
