# Structures Narratives Avancées

## Le Modèle de Transformation (Duarte)

### La Structure en S

```
                    NOUVEAU MONDE
                         /
                        /
         Résistance    /
              \       /
               \     /
                \   /
                 \ /
    MONDE ────────X───────────────→ TEMPS
    ACTUEL        Point de
                  bascule
```

**Éléments clés** :

1. **Le Monde Actuel** : Situation de départ
2. **Le Call to Adventure** : L'élément déclencheur
3. **La Résistance** : Obstacles et objections
4. **Le Point de Bascule** : Moment de transformation
5. **Le Nouveau Monde** : Vision du futur accompli

### Application en Présentation

| Moment | Monde Actuel | Nouveau Monde |
|--------|--------------|---------------|
| Émotion | Tension, inconfort | Espoir, motivation |
| Données | Problèmes actuels | Projections positives |
| Action | Ce qu'on fait | Ce qu'on fera |
| Ton | Sérieux, urgent | Inspirant, confiant |

## Les 7 Plots Universels (Christopher Booker)

Adapté aux présentations business :

### 1. Vaincre le Monstre
> "Nous affrontons un concurrent/problème puissant, et voici comment le vaincre"

**Structure** :
- Identifier le "monstre" (concurrent, marché, technologie)
- Montrer sa puissance (données menaçantes)
- Révéler notre arme secrète
- Visualiser la victoire

### 2. De la Misère à la Richesse
> "Nous étions en difficulté, maintenant nous prospérons"

**Structure** :
- Point bas (crise, échec)
- Tournant (décision, innovation)
- Ascension (progrès mesurables)
- Succès actuel (résultats)

### 3. La Quête
> "Nous cherchons quelque chose d'important et nous le trouverons"

**Structure** :
- L'objectif (vision, mission)
- Le voyage (étapes, défis)
- Les alliés (équipe, partenaires)
- La destination (but atteint)

### 4. Voyage et Retour
> "Nous avons exploré et voici ce que nous avons appris"

**Structure** :
- Le départ (exploration, R&D, pilote)
- La découverte (insights, données)
- Le retour (application pratique)
- La transformation (ce qui change)

### 5. Renaissance
> "Nous étions en déclin, mais nous renaissons"

**Structure** :
- Le déclin (problèmes, stagnation)
- La prise de conscience (diagnostic)
- La transformation (actions)
- Le renouveau (nouveaux résultats)

## Le Voyage du Héros en Business

### Version Simplifiée (12 → 5 étapes)

```
1. MONDE ORDINAIRE
   └── "Voici notre situation actuelle"

2. APPEL ET REFUS
   └── "Un défi se présente, avec ses risques"

3. TRAVERSÉE DU SEUIL
   └── "Nous avons décidé d'agir"

4. ÉPREUVES ET TRANSFORMATION
   └── "Voici ce que nous avons appris/fait"

5. RETOUR AVEC L'ÉLIXIR
   └── "Voici les bénéfices pour tous"
```

### Héros Possibles

| Héros | Utilisation |
|-------|-------------|
| L'entreprise | Transformation organisationnelle |
| Le client | Étude de cas, témoignage |
| Le produit | Lancement, évolution |
| L'équipe | Projet, réussite collective |
| L'industrie | Tendances, prospective |

## Structure TED (18 minutes)

### Le Framework de Chris Anderson

```
Minutes 0-3  : ACCROCHE
               └── Capturer l'attention immédiatement

Minutes 3-5  : CONTEXTE
               └── Établir le problème/opportunité

Minutes 5-12 : DÉVELOPPEMENT
               └── 3-4 points majeurs avec exemples

Minutes 12-15: SOLUTION/INSIGHT
               └── La révélation principale

Minutes 15-17: IMPLICATIONS
               └── Ce que ça change

Minutes 17-18: CALL TO ACTION
               └── Ce que l'audience peut faire
```

### Règles TED

1. **Une seule idée** qui vaut la peine d'être partagée
2. **Raconter des histoires** (minimum 65% du temps)
3. **Parler comme à un ami** (pas de jargon corporate)
4. **Révéler quelque chose de nouveau** (insight, données, perspective)

## Techniques Narratives Spécifiques

### In Medias Res

Commencer au milieu de l'action :

> ❌ "Bonjour, aujourd'hui je vais vous parler de..."
> ✓ "Il était 3h du matin quand le téléphone a sonné. Notre serveur principal venait de crasher."

### Le Cliffhanger

Créer du suspense avant une pause ou transition :

> "Et c'est là que nous avons découvert quelque chose d'inattendu... [pause] Mais avant de vous révéler quoi, regardons les données."

### La Règle de Trois

Organiser en trois points :

> "Trois raisons expliquent ce succès : l'équipe, la technologie, le timing."

### Le Callback

Revenir sur un élément mentionné plus tôt :

> [Début] "Souvenez-vous du client qui hésitait..."
> [Fin] "Ce client dont je parlais au début ? Il est devenu notre ambassadeur."

### L'Analogie

Rendre l'abstrait concret :

> "Notre infrastructure cloud est comme un système nerveux. Les données sont les impulsions électriques..."

### Le Contraste Dramatique

Opposer deux réalités :

> "Aujourd'hui : 5 jours pour traiter une commande. Demain : 5 heures."

## Structures par Type de Présentation

### Pitch Investisseurs (10 min)

```
1. Hook + Problème (2 min)
2. Solution + Démo (2 min)
3. Marché + Business Model (2 min)
4. Traction + Équipe (2 min)
5. Ask + Vision (2 min)
```

### Rapport Trimestriel (15 min)

```
1. Highlights du trimestre (3 slides)
2. Performance vs Objectifs (3 slides)
3. Challenges et Learnings (2 slides)
4. Outlook Q+1 (2 slides)
5. Asks et Décisions (2 slides)
```

### Proposition Commerciale (20 min)

```
1. Compréhension du contexte client (3 slides)
2. Défis identifiés (2 slides)
3. Notre approche (4 slides)
4. Preuves et références (3 slides)
5. Offre et investissement (3 slides)
6. Next steps (1 slide)
```

### Formation/Workshop (45 min)

```
1. Objectifs et contexte (5 min)
2. Concept 1 + Exercice (12 min)
3. Concept 2 + Exercice (12 min)
4. Concept 3 + Exercice (12 min)
5. Synthèse et Q&A (4 min)
```

## Gestion du Temps Narratif

### Pacing

| Moment | Rythme | Objectif |
|--------|--------|----------|
| Ouverture | Rapide | Capturer l'attention |
| Contexte | Modéré | Établir la base |
| Problème | Lent | Laisser résonner |
| Solution | Progressif | Construire l'argument |
| Call to Action | Énergique | Motiver |

### Points de Respiration

Intégrer des pauses :
- Après une statistique forte
- Avant une révélation
- Pour laisser une image s'imprimer
- Pour permettre des questions
