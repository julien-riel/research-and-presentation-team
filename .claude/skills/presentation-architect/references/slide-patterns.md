# Patterns de Slides

## Slides d'Ouverture

### Pattern: Title Slide Minimal

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│         TITRE EN MAJUSCULES             │
│         Sous-titre en minuscules        │
│                                         │
│                                         │
│                                         │
│                         Nom · Date      │
└─────────────────────────────────────────┘
```

**Usage** : Présentations formelles, kick-offs

## Slides de Contenu

### Pattern: Assertion-Evidence

```
┌─────────────────────────────────────────┐
│ Message clé en une phrase complète      │
├─────────────────────────────────────────┤
│                                         │
│                                         │
│         [VISUAL EVIDENCE]               │
│         Graphique, image,               │
│         ou diagramme                    │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

**Usage** : Présenter des données, prouver un point

### Pattern: Build Progressif

Slide 1:
```
┌─────────────────────────────────────────┐
│ Notre stratégie en 3 piliers            │
├─────────────────────────────────────────┤
│                                         │
│   ┌─────────┐                           │
│   │Pilier 1 │                           │
│   │Innovation│                          │
│   └─────────┘                           │
│                                         │
└─────────────────────────────────────────┘
```

Slide 2:
```
┌─────────────────────────────────────────┐
│ Notre stratégie en 3 piliers            │
├─────────────────────────────────────────┤
│                                         │
│   ┌─────────┐  ┌─────────┐              │
│   │Pilier 1 │  │Pilier 2 │              │
│   │Innovation│ │Expansion│              │
│   └─────────┘  └─────────┘              │
│                                         │
└─────────────────────────────────────────┘
```

Slide 3 (complète):
```
┌─────────────────────────────────────────┐
│ Notre stratégie en 3 piliers            │
├─────────────────────────────────────────┤
│                                         │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│   │Pilier 1 │  │Pilier 2 │  │Pilier 3 │ │
│   │Innovation│ │Expansion│ │Excellence│ │
│   └─────────┘  └─────────┘  └─────────┘ │
│                                         │
└─────────────────────────────────────────┘
```

**Usage** : Construire progressivement une idée complexe

### Pattern: Comparaison Côte-à-Côte

```
┌─────────────────────────────────────────┐
│ Avant vs Après la transformation        │
├──────────────────┬──────────────────────┤
│     AVANT        │       APRÈS          │
│                  │                      │
│  • 5 jours       │  • 5 heures         │
│  • 3 systèmes    │  • 1 plateforme     │
│  • 12% erreurs   │  • 0.5% erreurs     │
│                  │                      │
│  [Image sombre]  │  [Image lumineuse]  │
└──────────────────┴──────────────────────┘
```

**Usage** : Montrer le changement, vendre une solution

### Pattern: Timeline/Process

```
┌─────────────────────────────────────────┐
│ Notre parcours de transformation        │
├─────────────────────────────────────────┤
│                                         │
│   Q1         Q2         Q3         Q4   │
│   ┌──┐       ┌──┐       ┌──┐       ┌──┐ │
│   │▓▓│──────▶│▓▓│──────▶│▓▓│──────▶│▓▓│ │
│   └──┘       └──┘       └──┘       └──┘ │
│  Analyse   Design    Pilote    Scale   │
│                                         │
└─────────────────────────────────────────┘
```

**Usage** : Expliquer un processus, roadmap

## Patterns Avancés

### Pattern: Matrice 2x2

```
┌─────────────────────────────────────────┐
│ Priorisation des initiatives            │
├─────────────────────────────────────────┤
│              Impact élevé               │
│         ┌───────────┬───────────┐       │
│         │  Planifier│  Faire    │       │
│  Effort │  avec soin│  d'abord  │       │
│   élevé ├───────────┼───────────┤       │
│         │  Éliminer │  Quick    │       │
│         │           │  wins     │       │
│         └───────────┴───────────┘       │
│              Impact faible              │
└─────────────────────────────────────────┘
```

### Pattern: Funnel/Entonnoir

```
┌─────────────────────────────────────────┐
│ Notre pipeline commercial               │
├─────────────────────────────────────────┤
│                                         │
│      ┌─────────────────────┐ 1000       │
│      │      Prospects      │ leads      │
│      └──┬───────────────┬──┘            │
│         │   Qualifiés   │ 300           │
│         └──┬─────────┬──┘               │
│            │ Démos   │ 80               │
│            └──┬───┬──┘                  │
│               │ ✓ │ 25 clients          │
│               └───┘                     │
│                                         │
└─────────────────────────────────────────┘
```
