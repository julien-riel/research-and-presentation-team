# Techniques de Communication et Présentation Orale

## La Préparation (Avant)

### Checklist de Préparation

- [ ] Connaître son audience (noms, rôles, attentes)
- [ ] Maîtriser le contenu sans lire les slides
- [ ] Préparer les réponses aux questions probables
- [ ] Tester le matériel (projecteur, clicker, son)
- [ ] Répéter à voix haute (minimum 3 fois)
- [ ] Chronométrer chaque section
- [ ] Préparer un plan B si le temps est réduit

### La Règle 10-20-30 (Guy Kawasaki)

Pour les pitchs :
- **10 slides** maximum
- **20 minutes** de présentation
- **30 points** minimum pour la police

### Gestion du Stress

**Techniques physiques** :
- Respiration diaphragmatique (4-7-8)
- Power poses (2 minutes avant)
- Échauffement vocal

**Techniques mentales** :
- Visualisation positive
- Focus sur l'audience, pas sur soi
- Accepter l'imperfection

## Le Langage Corporel

### Position et Posture

```
✓ FAIRE                    ✗ ÉVITER
──────────────────────────────────────────
Pieds écartés (stable)     Croiser les bras
Épaules ouvertes           Se balancer
Regard vers l'audience     Tourner le dos
Mains visibles             Mains dans poches
Mouvement intentionnel     Tics nerveux
```

### Le Contact Visuel

**Technique du "Lighthouse"** :
- Balayer la salle comme un phare
- 3-5 secondes par personne/zone
- Inclure les coins de la salle
- Revenir aux décideurs clés

**Pour les présentations virtuelles** :
- Regarder la caméra (pas l'écran)
- Placer des yeux/photos près de la caméra
- Éviter de lire ses notes sur un autre écran

### Les Gestes

**Gestes efficaces** :
- Paumes ouvertes (honnêteté)
- Énumérer avec les doigts
- Délimiter des espaces (avant/après)
- Pointer vers les visuels

**À éviter** :
- Gestes répétitifs
- Mains qui cachent le visage
- Pointer du doigt vers l'audience
- Croiser les bras

## La Voix

### Les 4 Leviers

| Levier | Usage |
|--------|-------|
| **Volume** | Fort = important, Doux = confidentiel |
| **Rythme** | Rapide = énergie, Lent = gravité |
| **Ton** | Aigu = enthousiasme, Grave = autorité |
| **Pauses** | Avant/après points clés |

### Le Pouvoir de la Pause

- **Avant un chiffre important** : "Notre croissance a atteint... [pause]... 47%"
- **Après une question rhétorique** : "Et si on pouvait diviser ce temps par deux ? [pause]"
- **Pour laisser résonner** : "[message clé] [pause de 3 secondes]"

### Rythme Recommandé

- 120-150 mots/minute (conversation)
- 100-120 mots/minute (moments clés)
- Varier pour maintenir l'attention

## L'Engagement de l'Audience

### Techniques d'Interaction

**Questions rhétoriques** :
> "Qui ici a déjà vécu cette situation ?"
> "Imaginez un instant que..."

**Sondages à main levée** :
> "Levez la main si vous pensez que..."

**Interpellation directe** :
> "Marie, vous qui êtes en charge de..., qu'en pensez-vous ?"

**Exercices brefs** :
> "Prenez 30 secondes pour noter votre principale préoccupation"

### La Règle des 10 Minutes

Tous les 10 minutes maximum :
- Changer de rythme
- Poser une question
- Montrer un visuel fort
- Raconter une histoire
- Faire interagir

### Lire la Salle

**Signaux d'engagement** :
- Contact visuel
- Hochements de tête
- Prise de notes
- Questions

**Signaux de décrochage** :
- Regards vers les téléphones
- Postures avachies
- Chuchotements
- Bâillements

**Réactions** :
- Accélérer si désengagement
- Poser une question directe
- Changer de sujet/slide
- Proposer une pause

## Gestion des Questions

### Framework PREP pour les Réponses

- **P**oint : Répondre directement
- **R**aison : Expliquer pourquoi
- **E**xemple : Illustrer concrètement
- **P**oint : Reformuler la conclusion

### Types de Questions Difficiles

**La question hostile** :
> ✓ "C'est une préoccupation légitime. Voici comment nous l'adressons..."
> ✗ "Non, vous avez tort..."

**La question hors-sujet** :
> ✓ "Excellente question. Je propose d'en discuter après la session pour ne pas dévier..."

**La question à laquelle on ne sait pas répondre** :
> ✓ "Je n'ai pas cette information sous la main. Je vous reviens d'ici demain avec une réponse précise."

**La question piège** :
> ✓ "Je comprends l'angle, mais reformulons : la vraie question est..."

### Techniques de Clarification

- "Si je comprends bien, vous demandez..."
- "Pouvez-vous préciser ce que vous entendez par..."
- "Je veux m'assurer de bien répondre : votre question porte sur X ou Y ?"

## Storytelling Oral

### La Structure 3C

1. **Contexte** (15%) : Poser le décor rapidement
2. **Conflit** (70%) : Le cœur de l'histoire, la tension
3. **Conclusion** (15%) : La résolution et la leçon

### Techniques de Narration

**Montrer, pas dire** :
> ✗ "Le client était très satisfait"
> ✓ "Le client m'a appelé pour dire : 'C'est la première fois en 10 ans qu'un fournisseur tient ses promesses'"

**Détails sensoriels** :
> "Il était 6h du matin, le bureau était silencieux, seul le ronronnement des serveurs..."

**Dialogue** :
> "J'ai demandé : 'Qu'est-ce qui vous empêche de dormir ?' Il a répondu : 'Cette migration.'"

### Timing des Histoires

| Durée présentation | Nombre d'histoires | Durée par histoire |
|-------------------|-------------------|-------------------|
| 5 min | 1 | 1-2 min |
| 15 min | 2 | 2-3 min |
| 30 min | 3-4 | 2-4 min |
| 60 min | 4-6 | 3-5 min |

## Situations Spéciales

### Présentation Virtuelle

- Cadrage : visage + épaules
- Éclairage : face, pas de contre-jour
- Son : micro externe si possible
- Engagement : questions dans le chat
- Énergie : +20% par rapport au présentiel

### Présentation à un Décideur Pressé

- Commencer par la recommandation
- "Je suis ici pour vous demander X"
- Avoir les détails en backup
- Être prêt à écourter

### Présentation Technique à Non-Experts

- Analogies du quotidien
- Éviter le jargon
- Visualiser les concepts
- "Concrètement, ça veut dire..."

### Co-Présentation

- Définir clairement qui parle quand
- Transitions fluides ("Je passe la parole à...")
- Écoute active quand l'autre parle
- Pas d'interruption

## Gestion des Imprévus

| Imprévu | Réaction |
|---------|----------|
| Problème technique | Avoir un plan B, rester calme |
| Temps réduit | Connaître les slides à couper |
| Audience hostile | Écouter, valider, recadrer |
| Trou de mémoire | Pause, boire de l'eau, regarder les notes |
| Question déstabilisante | "Bonne question, laissez-moi y réfléchir..." |
