Crée une présentation qui explique comment fonctionne ce repo "Presentation Team" pour générer des présentations PowerPoint professionnelles avec Claude Code.

## Audience

- Développeurs curieux de Claude Code et des skills
- Équipes qui veulent automatiser la création de présentations
- Personnes intéressées par l'IA générative appliquée

# Theme
 Font: Open Sans
 Couleur du texte #333 sur fond blanc
 Couleur de la compagnie: #C00, dans ce cas, le texte est #000 pour les diapositives de sections
 Couleur du texte inversé: #fff sur fond noir, pour certains titres / haut de page

# Photos

Personnes qui travaillent, femme d'affaire fière, Homme utilisant un laptop (dans ce genre-là)

## Durée

10-12 slides, ~15 minutes de présentation

## Notes

C'est une présentation méta - elle démontre les capacités du système en s'expliquant elle-même. Les diagrammes générés par le système illustrent ce que le système peut faire.
